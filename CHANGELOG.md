# BethINI Pie — Changelog

## v3.5.2 — 2025-12-10
- Fixed Starfield INI path detection on Xbox Game Pass version
- Updated recommended shadow values for Skyrim AE 1.6.1170+
- Minor UI cleanup in the Visuals tab

## v3.5.0 — 2025-09-01
- Added Oblivion Remastered support
- New "Compare" feature: shows diff between current and vanilla defaults
- Backup manager now lists all backups with restore confirmation dialog

## v3.4.0 — 2025-06-15
- Added Starfield full support with StarfieldPrefs.ini editor
- Custom INI key editor now supports array values
- Preset system rewritten — presets are now per-game

## v3.3.2 — 2025-03-20
- Fixed incorrect default for bFloatPointRenderTarget in Skyrim LE
- Fixed crash when Documents path contained non-ASCII characters

## v3.3.0 — 2025-01-05
- Added real-time preview for shadow distance changes
- Improved auto-detection for non-standard Steam library paths

## v3.0.0 — 2024-08-20
- Complete UI rewrite using WPF
- Tabbed settings interface
- Auto-backup before every write operation

## v2.0.0 — 2024-01-10
- Added Fallout 4 and Fallout New Vegas support
- Preset system introduced

## v1.0.0 — 2023-06-01
- Initial release with Skyrim SE/LE support
